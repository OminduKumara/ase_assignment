# SLIIT Tennis Management System - Frontend

This is the frontend client for the SLIIT Tennis Management Web Application, built with React.js and Vite.

## Prerequisites
* Node.js (LTS version recommended)
* Git

## Getting Started

1. **Navigate to the client directory:**
   ```bash
   cd client

2. **Install dependencies:**
  npm install

3. **Run the developments server:**
  npm run dev

The application will be available at http://localhost:5173/.
